# HRT Datathon 2026: Winning Solution

This repo contains the winning submission for the [HRT Datathon Challenge](https://www.linkedin.com/posts/andreas-akarepis-b80874200_datathon2026-quantfinance-machinelearning-activity-7453062508501987328-V64m), organized by the [ETHZ Data Analytics Club](https://www.analytics-club.org/) in collaboration with HRT.

<br/>


# Installation & execution

## Create virtual environment + install dependencies

```bash
uv venv
source .venv/bin/activate
uv sync
```


## Run predictor

```bash
uv run strategy.py
```
